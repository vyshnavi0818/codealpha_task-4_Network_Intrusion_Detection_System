# Network Intrusion Detection System (NIDS)

## Objective

The objective of this project is to implement a **Network Intrusion Detection System (NIDS)** using **Snort 3** to monitor network traffic and detect ICMP (Ping) packets by creating and testing a custom detection rule.

## Tools Used

* Kali Linux
* VMware Workstation
* Snort 3
* Nano Text Editor
* Terminal

## Features

* Installs and configures Snort 3
* Monitors live network traffic
* Detects ICMP (Ping) packets
* Uses a custom Snort rule for intrusion detection
* Generates real-time alerts for detected packets
* Validates Snort configuration successfully

## Output

The system successfully monitors network traffic and generates alerts whenever ICMP packets are detected using the custom Snort rule.
